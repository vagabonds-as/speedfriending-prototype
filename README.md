# Speedfriending — Klikkbar prototype

Dette er designprototypen for Speedfriending-appen — en klikkbar, interaktiv demo som viser hvordan appen ser ut og føles før produksjonskoden skrives.

## Nøkkelinnhold

- **6 Storylines** — kuraterte brukerreiser (Karins år, Bli ambassadør, HR-Ingrid, Aperol, Dating, Viral-loop)
- **10 Quick Flows** — raske hopp til Social, Business, Dating, B2B, Sponsor, Ambassadør, Avanserte, Wrapped, Nettside
- **100+ skjermer** som dekker hele appens overflate
- **Live App mode** (kommer) — kobler prototypen til ekte data

## Kjør lokalt

```bash
python3 -m http.server 8000
```

Åpne deretter [http://localhost:8000/Prototype.html](http://localhost:8000/Prototype.html).
